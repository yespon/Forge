# Bootstrap Skill

This skill helps with initial setup and configuration of the Forge environment.

## Capabilities

- Verify system configuration and dependencies
- Guide users through first-time setup
- Provide help with common configuration tasks

## Usage

When the user asks for help getting started, guide them through:
1. Checking environment variables are set
2. Verifying database connectivity
3. Testing the agent runtime
